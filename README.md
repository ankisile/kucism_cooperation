## 1. 웹페이지 기능

‘노페인 노게인’은 머신러닝을 통해 카톡 대화 내용을 분석하여 팀원 기여도를 측정 및 관리해주는 서비스다. 분석하고 싶은 대화방의 대화 내용을 TXT 파일로 내보내기 하고, 그 파일을 업로드하면 대화 내용을 분석 및 저장해준다.

    1. 로그인, 회원가입
    2. 프로필, 메달, 
    3. 파일 업로드
    4. 분석기록 테이블
